This is an aside. It's often used for side content, like a sidebar or a pull-quote. It can contain any type of content, but it's often used for content that is tangentially related to the main content. For example, in a blog post about cooking, an aside might contain a brief history of a particular ingredient, or a list of other recipes that use that ingredient.
